# Tangle

Untangle/Planarity puzzle game. Drag vertices to eliminate edge crossings across 100 bundled procedural levels. Phaser 4 + React 19 + Capacitor 8.

## Stack

- **Phaser 4.0.0** for the game canvas
- **React 19 + TypeScript** for the UI shell
- **Vite 6**
- **Tailwind CSS 3.4** (React UI only — do NOT style Phaser game objects with Tailwind)
- **Vitest 3** for unit tests
- **Playwright** for E2E smoke
- **ESLint 9 flat + Prettier 3**
- **Capacitor 8** wraps for iOS/Android (generated per-game via `npx cap add`)

## Architecture

```
src/
  game/
    EventBus.ts              # Typed hand-rolled EventEmitter. Extend EventMap when adding events.
    main.ts                  # Phaser config + startGame()
    scenes/
      BootScene.ts           # Waits for fonts (max 1.5s), starts Game
      GameScene.ts           # Blank scene; placeholder text
    rendering/tokens.ts      # Colors/layout constants shared with Tailwind vars
    domain/                  # Pure TS. NO Phaser, NO React, NO DOM, NO localStorage.
      types.ts               # Vec2, Vertex, Edge, Graph, Level, GameState
      rng.ts                 # mulberry32 seedable PRNG + seeded shuffle
      geometry.ts            # distanceSq, clamp, clampPoint, segmentsIntersect
      crossings.ts           # countCrossings, edgesIncidentTo, crossingsInvolving
      generator.ts           # Delaunay-based level generator (d3-delaunay)
      solver.ts              # Optimal-move estimate for star ratings
      levels.ts              # Loader wrapper around the generated data
      levels.generated.ts    # AUTO-GENERATED by scripts/generate-levels.ts — do not edit
  platform/
    native.ts                # Capacitor bootstrap (status bar, orientation, splash, back)
    share.ts                 # Share wrapper (native → web-share → clipboard)
  storage/
    persistence.ts           # Generic safeGet/safeSet/loadJson/saveJson
    ads.ts                   # Dev-stub AdProvider. Swap for AdMob when shipping.
  telemetry/
    analytics.ts             # Dev noop. setAnalyticsProvider() to swap.
    crash.ts                 # Dev noop. setCrashReporter() to swap.
  styles/globals.css         # Tailwind + self-hosted fonts + safe-area utilities
  ui/
    App.tsx                  # Single-screen container
    PhaserGame.tsx           # Mounts Phaser, wires EventBus
    components/ErrorBoundary.tsx
  main.tsx                   # React entry: bootstrap + StrictMode + ErrorBoundary
```

### Layering rules

- `domain/` is pure TypeScript. No Phaser, no React, no DOM, no localStorage, no network. Fully deterministic.
- `storage/` wraps platform side-effects. Depends on `domain/` for types.
- Phaser code (`src/game/` outside `domain/`) renders and owns input but never owns state. It reads from React state and emits events via `EventBus`.
- React code (`src/ui/`) owns navigation and chrome, subscribes to `EventBus` via `PhaserGame`.

## Conventions

- Single quotes, 2-space indent, semicolons, trailing commas, 100-char width. Enforced by Prettier.
- `import type` for type-only imports; ESLint rule `consistent-type-imports` flags violations.
- Strict TypeScript (`strict` + `noImplicitOverride` + `noImplicitReturns` + `noUnusedLocals` + `noUnusedParameters`).
- No comments on obvious code. Comment only non-obvious WHY.
- No `console.log` in committed code outside `scripts/` and tests.
- Tests live in `__tests__/` sibling folders.

## Renaming for a new game

`npm run rename -- --name "My Game" --id "com.me.mygame" --slug "my-game"` updates:
- `package.json` (`name`)
- `capacitor.config.ts` (`appId`, `appName`, `Preferences.group`)
- `index.html` (title, apple meta, description)
- `public/manifest.webmanifest` (`name`, `short_name`, `description`)

After rename, regenerate native projects:
```
rm -rf ios android
npx cap add ios
npx cap add android
```

## Swapping stubs for production

| Stub | Swap for |
|---|---|
| `telemetry/analytics.ts` noop | Real provider implementing `AnalyticsProvider`, installed via `setAnalyticsProvider()` in `main.tsx` before `initAnalytics()` |
| `telemetry/crash.ts` noop | Real reporter implementing `CrashReporter`, installed via `setCrashReporter()` before `initCrash()` |
| `storage/ads.ts` `DevStubAdProvider` | See inline comment — install `@capacitor-community/admob` and branch in `selectProvider()` |

## Out of scope for the template
